# 🌐 Multi-Tool Hub

> A modern and responsive web application that combines powerful text translation and secure password generation in one place.

## ✨ Overview

Multi-Tool Hub is a lightweight and user-friendly web application built with React and Vite. It provides two practical utilities in a single interface:

🌍 **Language Translator** – Translate text into multiple languages using RapidAPI.

🔐 **Password Generator** – Generate strong and secure passwords instantly for enhanced online security.

The application features a clean UI, responsive design, and fast performance, making it suitable for both desktop and mobile users.


## 🚀 Features

### 🌍 Translator

* Translate text between multiple languages
* Fast and accurate translations
* Powered by RapidAPI
* Simple and intuitive interface
* Responsive design for all devices


### 🔐 Password Generator

* Generate strong random passwords
* Adjustable password length
* Includes uppercase letters
* Includes lowercase letters
* Includes numbers
* Includes special characters
* One-click password generation


### 🎨 User Experience

* Modern UI with Tailwind CSS
* Mobile-friendly responsive layout
* Fast loading with Vite
* Clean and organized component structure
